A bio-inspired, Unix-inspired paradigm for a hybrid human–agent world
where legacy code coexists with new code, machines compose with machines,
and humans orchestrate the whole.

**All you need is a `.proto` — illuminated by four facets:**

1. **Contract** — the `.proto` itself: language-agnostic, universal
2. **Code** — a thin native API for in-process composition
3. **CLI** — a terminal bridge for humans, scripts, and CI
4. **Tests** — the executable proof that the contract holds

Legacy code, new code, humans, agents: same interface, every facet.
